폼(Form) 태그

주요 속성 - action: 데이터를 전송할 url 주소입니다.
- method: 데이터를 전송할 방법입니다.
- ectype: 데이터를 전송할 때 전송 인토딩 형식을 지정합니다.

input 태그

홀 태그 형식. 입력상자를 만드는 태그입니다. type의 속성에 따라 입력칸, 선택박스(라디오 버튼, 체크 박스), 버튼을 표시할 수 있습니다.

주요 속성 - type: 입력 형태를 지정합니다.
- name: 입력 값의 이름을 지정합니다.
- value: 입력 상자의 값을 지정할 수 있습니다..


radio1 radio2 radio3
checkbox1 checkbox2 checkbox3

selection 태그

드롭다운 선택 박스를 만듭니다. option 태그를 사용해서 선택지를 추가합니다.

textarea 태그

긴 글을 입력 받도록 해주는 입력 칸을 만든다. 쌍 태그로 만들어지고 태그 사이에 긴 값을 넣을 수 있습니다.

label 태그

사용자에게 각 입력 칸을 설명해주는 레이블을 표시할 수 있습니다. 입력 태그에 id 속성으로 지정 된 값을 for 속성으로 넣어주면 그 입력 태그를 수식해주게 됩니다.

fieldset, legend 태그

여러 입력 태그를 하나의 그룹으로 묶고 원하는 제목을 달 수 있습니다.
